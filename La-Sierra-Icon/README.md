## How to install:

### First:
Download:

     $ git clone https://github.com/btd1337/La-Sierra-Icon-Theme ~/.icons/La-Sierra

### Second:
Set:

     $ gsettings set org.gnome.desktop.interface icon-theme "La-Sierra"
